In this lab, we will exploit a vulnerable web server via SQL Injection that allows access to a PHP eval function. We will then identify a vulnerable setuid Screen binary and modify the privilege escalation exploit to suit the target environment.

After completion of this lab, learners will be able to:

- Enumerate open services to identify potential SQL injection vulnerabilities in the web application.
- Exploit the SQL injection vulnerability to inject PHP code for remote code execution.
- Establish a reverse shell as the apache user to gain a foothold on the target system.
- Identify and analyze the setuid-enabled Screen binary for privilege escalation opportunities.
- Exploit the Screen binary to gain root access on the target system.
